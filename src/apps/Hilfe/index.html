App öffnen

Gehen Sie, wenn Sie eine App geöffnet haben, auf „Start", sodass Sie Ihren Startbildschirm sehen und gehen Sie dann mit dem Mauszeiger auf das Bild von der App und klicken Sie auf Ihre linke Maustaste.

Zum Startbildschirm zurückkehren

➔ Machen Sie einen Linksklick auf den großen grünen Knopf, links-unten in der Ecke, auf dem „Start" steht.

Dateienmanager

➔ Dateien öffnen: Klicken Sie auf die Datei mit der linken Maustaste, oder Rechtsklick auf eine freie Fläche und dann mit einem Klick mit der limken Maustaste auf „Öffnen".

➔ Dateien erstellen: Rechtsklick auf eine freie Fläche und dann klicken Sie mit der linken Maustaste auf „Neue Datei".

➔ Ordner erstellen: Rechtsklick auf eine freie Fläche und dann klicken Sie mit der linken Maustaste auf „Neuer Ordner".

➔ Umbenennen: Rechtsklick auf eine Datei und dann Linksklick auf „Umbenennen".

➔ Datei löschen: Machen Sie einen Rechtsklick auf eine Datei und machen sie dann einen Linksklick auf cLöschen".

Taschenrechner

➔ Zahlen bzw. Zeichen eingeben: Gehen Sie mit dem Cursor auf die Zahl oder das Zeichen, das Sie eingeben wollen und klicken sie auf Ihre linke Maustaste.

➔ Rechenzeichen:

⮑ Plus: „+"

⮑ Minus: „-"

⮑ Mal: „*"

⮑ Durch: „/"

⮑ Modulo: „%"

➔ Eingabe löschen: Betätigen Sie mit einem Linksklick den roten Knopf mit dem „C".

Kalender

➔ Ereignis erstellen: Klicken Sie mit der linken Maustaste auf den Tag (z.B auf die 20, die unter der Überschrift November steht), bei dem Sie das Ereignis hinzufügen wollen. Machen Sie dann wieder einen Linksklick auf „Ereignis erstellen".

➔ Ereignis löschen: Klicken Sie mit der linken Maustaste auf den Tag (z.B auf die 20, die unter der Überschrift November steht) bei dem Sie das Ereignis löschen wollen und machen Sie dann wieder einen Linksklick auf „Ereignis löschen".

Kontakte

➔ Kontakt hinzufügen: Machen Sie einen Rechtsklick auf den grünen Knopf unten-rechts in der Ecke, füllen Sie die Daten ein und klicken sie mit der linken Maustaste auf „Kontakt hinzufügen". Wenn Sie dan Vorgang doch abbrechen wollen machen sie einen Linksklick auf „Abbrechen".

➔ Kontakt löschen: Klicken sie mit der linken Maustaste auf den Knopf, rechts neben dem Kontakt, den Sie löschen wollen.

➔ Kontakt bearbeiten: Klicken Sie auf den grünen Knopf, rechts neben dem Kontakt, den Sie bearbeiten wollen und füllen Sie die Felder so aus, wie Sie es haben wollen. Wenn Sie fertig sind, klicken Sie mit der linken Maustaste auf „OK". Wenn Sie es doch Abbrechen wollen machen sie einen Linksklick auf „Abbrechen".

(➔ Die Kontakte werden automatisch gespeichert!)

Radio

➔ Radio anmachen: Machen Sie einen Linksklick auf das weiße Dreieck, welches um 90°, so dass es nach rechts zeigt, unter dem Radiosender, den Sie hören möchten.

➔ Radio pausieren: Machen Sie einen Linksklick auf die parallelen, weißen und senkrechten Striche, unter dem Radiosender, den du pausieren möchtest.

➔ Radio stoppen: Machen Sie einen Linksklick auf das weiße Viereck, rechts neben dem langen Strich, unter dem Radiosender, den du stoppen möchtest.

Uhr

➔ Timer / Stoppuhr / Wecker auswählen: Klicken Sie mit der linken Maustaste auf den Knopf, bei dem steht was Sie haben wollen.

➔ Timer:

⮑ Timer einstellen: Geben Sie ihre Sekunden, Minuten, Stunden und Tage, die Sie haben wollen, durch die Pfeile rechts in dem Kasten ein oder benutzen Sie dafür ihre Tastatur, indem Sie ein Linksklick in das Feld machen und dann auf ihrer Tastatur die zahlen eingeben. (Wenn Sie keine Sekunden / Minuten / Stunden / Tage wollen können Sie das Feld auch leer lassen.)

⮑ Timer starten: Klicken Sie mit der linken Maustaste auf den grünen Knopf mit „Starten", in der rechten unteren Ecke vom grauen Rechteck.

⮑ Timer zurücksetzen: Klicken Sie mit der linken Maustaste auf den roten Knopf mit „Abbrechen", in der linken unteren Ecke vom grauen Rechteck.

➔ Wecker:

⮑ Wecker einstellen: Geben Sie ihre Sekunden, Minuten, Stunden, Tage, Jahre, die Sie haben wollen, durch die Pfeile rechts in dem Kasten ein oder benutzen Sie dafür ihre Tastatur, indem Sie ein Linksklick in das Feld machen und dann auf ihrer Tastatur die zahlen eingeben. (Wenn Sie keine Sekunden / Minuten / Stunden / Tage / Jahre wollen können Sie das Feld auch leer lassen.) Bei den Monaten müssen Sie auf das Feld einen Linksklick machen und dann den Monat mit einem weiteren Linksklick auswählen.

⮑ Wecker starten/abbrechen: (Genauso wie bei dem Timer)

➔ Stoppuhr:

⮑ Stoppuhr stoppen: Machen Sie einen Linksklick auf den roten Knopf auf dem „Stopp" steht.

⮑ Stoppuhr neustarten: Klicken Sie mit der linken Maustaste auf den grünen Start-Knopf. (Dadurch wird die Stoppuhr auf null gesetzt und dannach gestartet.)

⮑ Stoppuhr weiterlaufen lassen: Klicken sie mit der linken Maustaste auf den grünen Knopf mit „Weiter".
