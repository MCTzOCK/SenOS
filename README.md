# Building from source
## My Environment:
- Windows 10 (Pro)
- Node v14.15.1
- npm 7.5.3

## Building:
```bash
# clone repo
git clone https://github.com/MCTzOCK/SenOS.git && cd SenOS
# install dependencies
npm install

# build
# for windows
npm run make
# for linux
npm run makeLX
```
You will find the generated files in the ``out`` directory.
